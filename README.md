# WATER POTABILITY PREDICTION USING MACHINE LEARNING

# DESCRIPTION
* Analyzed the water potability dataset to assess and predict water safety using machine learning techniques.

  # OBJECTIVES
  To identify the most effective supervised learning model for accurately classifying water as potable or non-potable.

  # METHODLOGY
  * Performed Exploratory Data Analysis(EDA), handled missing values, applied feature scaling, and trained multiple supervised     learning models including KNN, Decision Tree, and XGBoost.
 
  # KEY RESULTS
  * Evaluated models using accuracy and AUC score, with the best-performing model achieving approximately 0.6631 accuracy and       0.6832 AUC score.

  # IMPACT
  Provides a comparative framework for selecting optimal models in water quality prediction and highlights areas for further       model improvement.
 # TOOLS $ TECHNOLOGIES
 Python, Pandas, Numpy, Scikit-learn, XGBoost, Matplotlib, Seaborn
